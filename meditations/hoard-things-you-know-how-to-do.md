# Hoard Things You Know How To Do

When you find a prompt, pattern, or technique that reliably works — save it. Don't rediscover it next time.

The learning curve for any new technology used to be the blocker (2-3 months to learn AppleScript = never learned it). Now the cost of trying something new is near zero. But the output of that — working prompts, reliable approaches, solved patterns — has value and should be kept.

Build a personal library. Templates, prompts that find specific bug classes, agent configurations that work, tool combos that consistently deliver. Each one is a force multiplier you can reach for again.

The people getting the most out of agents aren't starting from scratch each time. They're accumulating a kit.
