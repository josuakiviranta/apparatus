# Code Is Now Cheap — Rethink Everything Around It

Your intuitions about what's worth building are wrong. 25 years of experience calibrating "this will take 2 weeks, not worth it" no longer applies. The reason something used to take 2 weeks was all the crafty coding work — the AI now covers that.

Throw tasks at agents that you don't think they'll be able to do. Every now and then they surprise you.

Downstream: verbose test suites, 100+ tests on a small library, boilerplate-heavy patterns — all previously uneconomical — are now fine. Code is cheap. Write more of it. Throw it away if needed.

The bottleneck has moved. It's no longer implementation. Rethink where the real constraints are: proving ideas, QA, spec quality, product decisions.
